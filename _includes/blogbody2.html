Assignment: Create a function that checks if firstName is an actual contact's first name and the given property prop is a property of that contact, returning value of that property if both are true. Additionally, if firstName does not correspond to any contacts then return "No such contact" and if prop does not correspond to any valid properties then return "No such property".

Or, in plain English, is that person in our list and, if so, do they like these things? Either way, give them the corresponding messages above.

The breakdown and pseudo-code:

Loop through the contacts, checking firstName against the contacts listed.

  • If there is not a match, return "No such contact".
  • Else, proceed on to check on the prop.

Loop through the each contacts object checking if the prop is there.

  • If there is a match, return value of the matched prop.
  • Else, return "No such property".


 if(firstName IS-NOT-IN contacts){
  return "No such contact"
 }else{
  for(each "contact" IN contacts){
   if(prop IS-IN "contact"){
    return value of prop
   }else{
    return "No such property"
   }
  }
 }
